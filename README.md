# EDA---Telemetria-Veicular
Análise Exploratória de Dados Aplicada na Telemetria de Veículos de uma Frota


### Privacidade e anonimização dos dados

Este projeto foi desenvolvido com base em um cenário real de telemetria veicular, porém os dados disponibilizados neste repositório são sintéticos e/ou anonimizados.

Foram removidos ou substituídos todos os identificadores diretos e indiretos, incluindo placas, motoristas, clientes, endereços, coordenadas exatas, códigos internos, dispositivos de rastreamento e qualquer informação que pudesse permitir a identificação da empresa, dos condutores, dos clientes ou das rotas operacionais.

As datas, localizações e identificadores foram generalizados, mascarados ou sintetizados, preservando apenas padrões estatísticos necessários para demonstrar as etapas da análise exploratória de dados.



### Contexto

A telemetria é uma tecnologia que permite coletar e transmitir dados à distância para uma central que processa esses dados e os transforma em painéis de monitoramento.

A telemetria aplicada em frotas veiculares tem por objetivo o monitoramento dos dados de veículos sobre: condução, consumo, segurança e auxiliam na identificação de possíveis infrações no trânsito.

Neste exemplo de projeto proposto, a análise exploratória surgiu da necessidade de monitorar o estado dos veículos e o comportamento dos motoristas a fim de evitar possíveis danos aos veículos e antever problemas de mal comportamento dos motoristas no trânsito.

A princípio, o próprio software da empresa de telemetria foi suficiente para acompanhar as frotas. Entretanto haviam problemas de comportamento do motorista e de eventos que poderiam levar a infrações no trânsito que não estavam sendo acompanhados de forma histórica. Por isso, foi solicitada uma análise desses problemas com base nos últimos 4 meses para entender onde estariam os principais problemas de comportamento do motorista e sua direção.
